# Architecture du Système de Contrôle (DCS)
Système : ABB Ability™ Symphony Plus.

## Composants Réseau
- **Réseau Local (Control Bus)** : Ethernet Redondant 1 Gbps.
- **Contrôleurs (HPC)** : 2 par bloc (Maître/Esclave).
- **Postes Opérateurs (OWS)** : 4 en salle de commande.
- **Serveur d'Archives** : Stockage historique 5 ans.

## Protocoles supportés
- Modbus TCP/RTU (Auxiliaires tiers).
- Profibus DP (Instrumentation intelligente).
- OPC UA (Export données vers le siège).
